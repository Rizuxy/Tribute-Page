Presiden Joko Widodo

  • 2014 - Meluncurkan program Kartu Indonesia Sehat (KIS) untuk memperluas akses layanan kesehatan.
  • 2015 - Membangun Tol Trans-Jawa untuk meningkatkan konektivitas antar kota di Pulau Jawa.
  • 2016 - Menginisiasi Program Dana Desa untuk mempercepat pembangunan di desa-desa.
  • 2017 - Meresmikan operasionalisasi Kereta Bandara Soekarno-Hatta untuk memudahkan akses transportasi.
  • 2018 - Meluncurkan Program Kartu Pra Kerja untuk meningkatkan keterampilan tenaga kerja.
  • 2019 - Meresmikan Moda Raya Terpadu (MRT) Jakarta sebagai transportasi massal modern pertama di Indonesia.
  • 2020 - Memimpin penanganan pandemi COVID-19 dengan berbagai kebijakan kesehatan dan ekonomi.
  • 2021 - Mengakselerasi pembangunan infrastruktur digital, termasuk jaringan 5G.
  • 2022 - Meluncurkan program vaksinasi massal COVID-19 untuk seluruh masyarakat Indonesia.
  • 2023 - Mempercepat pembangunan Ibu Kota Negara (IKN) Nusantara di Kalimantan Timur.
